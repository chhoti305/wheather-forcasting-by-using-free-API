# 🌤️ Skyglass - Weather Intelligence Dashboard

Skyglass is a modern and elegant weather application that provides real-time weather intelligence for any location worldwide. Built using pure HTML, CSS, and JavaScript, it features a premium glassmorphism-inspired UI and displays comprehensive weather information including temperature, air quality, wind conditions, humidity, UV index, and more.

---

## ✨ Features

- 🔍 Search weather by city, region, or coordinates
- 🌡️ Real-time temperature and "feels like" temperature
- ☁️ Current weather conditions with dynamic icons
- 💧 Humidity monitoring
- 🌬️ Wind speed and direction
- 👀 Visibility information
- 📈 Atmospheric pressure data
- ☀️ UV Index with severity classification
- ☁️ Cloud cover percentage
- 🌍 Air Quality Index (AQI) with health indicators
- 🕒 Local time display
- 📍 Geographic coordinates
- 🎨 Modern responsive UI design
- ⚡ Fast and lightweight implementation
